# MyBlog

## Installation

```cmd
docker-compose build
docker-compose run web rake db:create
docker-compose up
```

[Quickstart: Compose and Rails](https://docs.docker.com/compose/rails)

## Credentials

```cmd
docker-compose run web rails credentials:edit
```
